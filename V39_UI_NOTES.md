# v39-production-card

Что исправлено относительно v38:

- Сохранена модульная/Vite-структура v38 и `public/runtime-config.js`.
- Карточка поста возвращена к финальной компактной логике из последних single-file версий.
- Дата в карточке поста снова открывает календарь-поповер с месяцем, слотами и хэштегами.
- Цветовая логика календаря даты:
  - зелёный фон дня — текущий хэштег свободен;
  - красный фон дня — текущий хэштег уже стоит в этот день;
  - фиолетовые плашки внутри дня — другие публикации/слоты.
- Фиолетовый больше не становится фоном дня.
- Текст поста и заметки сжаты по высоте.
- Блок материалов сжат.
- Сохранение `post_date`, `day_name`, `week_no`, `slot_order` осталось через общий `savePost()` и Supabase payload.
- Проверка `npm run check` проходит.

Контрольная метка в шапке: `v39-production-card`.
Контрольная метка в календаре даты: `v39-card`.
