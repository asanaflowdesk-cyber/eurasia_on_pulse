# v45 global filters and calendar week numbering

## Что изменено

- Фильтры верхней панели применяются ко всем рабочим вкладкам:
  - План: период, блок, статус, поиск.
  - Календарь: период, блок, статус, поиск; в календаре видны только записи, проходящие фильтр.
  - Банк идей: период, блок, статус, поиск.
  - Материалы: фильтруются по своим полям и по связанному посту; если выбран период/блок/статус, материалы без привязки к посту скрываются.
  - Справочники: поиск работает по значениям; фильтр блоков применяется к хэштегам.
  - Доступы: поиск работает по email, имени, роли и видимости.
  - Логи: период и поиск работают по логам.

- Нумерация недели заменена на календарную ISO-неделю: КН 1–53.
- Поле week_no по-прежнему сохраняется в pulse_posts, но теперь содержит календарную неделю даты публикации.
- Датированные посты считаются контент-планом даже если статус = idea; идея как отдельная сущность — это пост без даты.
